SQL 語法筆記 - WHERE 子句用法

By Infinity 2023-12-02

WHERE 子句是在 SQL 語句中用來篩選檢索條件的部分。當你想要檢索表中滿足特定條件的資料時，就會使用 WHERE。以下是 WHERE 子句的基本用法：

SELECT column1, column2, ...
FROM table_name
WHERE condition;

這裡是一些 WHERE 子句的使用範例：

  1. 基本的相等比較：
SELECT * FROM employees
WHERE department = 'IT';
  1. 不等於：
SELECT * FROM products
WHERE price <> 50;
  1. 範圍條件：
SELECT * FROM orders
WHERE order_date BETWEEN '2023-01-01' AND '2023-02-01';
  1. 使用 IN 運算子：
SELECT * FROM customers
WHERE country IN ('USA', 'Canada', 'Mexico');
  1. 模糊查詢（使用 LIKE）：
SELECT * FROM products
WHERE product_name LIKE 'App%';

這裡 % 是一個萬用字符，代表零個或多個字符。

  1. 邏輯運算符（AND, OR, NOT）：
SELECT * FROM employees
WHERE department = 'HR' AND salary > 50000;

你可以使用 AND、OR 和 NOT 來結合多個條件。

  1. 使用通配符 _：
SELECT * FROM customers
WHERE contact_name LIKE 'John _oe';

這裡 _ 代表單個字符，所以這個查詢將匹配 “John Doe”、“John Roe” 等。

這僅僅是 WHERE 子句的一些基本用法，還有更複雜和高級的用法，可以通過組合多個條件來進行更精確的篩選。請注意，WHERE 子句可以應用於 SELECT、UPDATE、DELETE 等 SQL 語句中，以根據需要檢索、更新或刪除資料。
